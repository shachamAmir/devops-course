# DevOps Course

Brief course materials and exercises for learning core DevOps concepts.

## Overview

- Purpose: Hands-on introduction to DevOps practices, tools, and workflows.
- Topics: CI/CD, containerization, infrastructure as code, monitoring, and automation.

## Structure

- `labs/` – Hands-on exercises
- `examples/` – Sample configuration and scripts
- `notes/` – Lecture notes and references

## Quick Start

1. Clone the repository:

   git clone <repo-url>

2. Open the lab of interest under `labs/` and follow the instructions.

## Prerequisites

- Git
- Docker (optional)
- A code editor (VS Code recommended)
